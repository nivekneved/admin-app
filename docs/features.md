# Travel Lounge Admin Panel Features

## Core Modules

- **Dashboard**: Real-time overview of bookings, revenue, and customer activity.
- **Customers**:
  - Management of user profiles and contact history.
  - Dedicated "view customer history" feature for bookings/orders.
- **Invoices**: Generation and management of billing documents.
- **Products & Services**: Full CRUD operations for all travel offerings.
- **Bookings**: Automated tracking and status management for all reservations.
- **News Manager**: Component for publishing and managing company updates.
- **Team**: Staff management and role assignment.

## Interface & Quality

- **Unified Brand Design**: Strategic use of the core red theme throughout the UI.
- **Responsive Tables**: Mobile-optimized stacking tables for effortless navigation on any device.
- **Performance**: Code splitting and error boundaries for maximum stability.
- **Security**: Supabase RLS and input sanitization (React Hook Form + Zod).
