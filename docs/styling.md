# Admin App Styling Guidelines

## Color Palette

- **Primary**: Brand Red (Tailwind `red-600`) - Used for headers, primary buttons, and active states.
- **Secondary**: Slate (Tailwind `slate-900`) - Used for sidebar and text.
- **Accent**: White/Light Gray - Used for card backgrounds and navigation surfaces.

## Component Standards

- **Tables**: Must be responsive and stack vertically on mobile.
- **Forms**: Use `react-hook-form` with `Zod` validation.
- **Icons**: Use `lucide-react` for a consistent, professional icon set.
- **Gradients**: Subtle linear gradients (e.g., `bg-gradient-to-r from-red-600 to-red-700`) for high-impact headers.
