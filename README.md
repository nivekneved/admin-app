# Travel Lounge - Admin Panel

A high-performance command center for managing Travel Lounge operations. Built for speed, precision, and visual consistency.

## 🚀 Core Capabilities

- **Unified Branding**: Strategic application of the core red theme across all modules.
- **Responsive Tables**: Mobile-first vertical stacking for all data views.
- **Customer Intelligence**: Detailed history tracking for bookings and orders.
- **News Manager**: Integrated system for publishing company updates.
- **Real-time Metrics**: Dynamic dashboard for instant operational awareness.

## 💻 Setup

1. **Install Dependencies**: `npm install`
2. **Environment**: Create a `.env` file (see `.env.example`)
3. **Run Dev**: `npm run dev`
