# Travel Lounge Admin Panel

This is the admin panel for managing travel lounge operations.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file with your Supabase configuration (see `.env.example`)

3. Run the development server:
```bash
npm run dev
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Supabase
- React Query
- Chart.js
